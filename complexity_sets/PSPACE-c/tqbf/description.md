<p>Il <strong>True Quantified Boolean Formula (TQBF)</strong> è QBF ristretto a formule in forma prenessa normale (tutti i quantificatori all'esterno) con una matrice in CNF. È il problema PSPACE-completo prototipico.</p><p>Stockmeyer e Meyer (1973) hanno dimostrato che TQBF è PSPACE-completo. La prova usa il teorema di Savitch per mostrare PSPACE ⊆ TQBF e un algoritmo PSPACE diretto per la valutazione di TQBF.</p>
