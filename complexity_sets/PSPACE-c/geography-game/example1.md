<p><strong>Esempio: gioco di geografia con le città</strong></p><p>Ogni giocatore nomina una città che inizia con l'ultima lettera della città precedente; nessuna città può essere ripetuta. Modellato come grafo diretto: nodi = città, archi = (A,B) se l'ultima lettera di A = la prima lettera di B.</p><p>Il primo giocatore vince sse il nodo di partenza ha una strategia vincente — decidibile in spazio polinomiale tramite ricerca nell'albero di gioco.</p>
