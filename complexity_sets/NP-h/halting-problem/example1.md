<p><strong>Esempio: ciclo infinito e congettura di Collatz</strong></p><p>Programma: while (collatz(n) != 1) { n = collatz_step(n); }. Determinare se questo termina per tutti gli input equivale alla congettura di Collatz — un problema aperto in matematica.</p><p>Gli analizzatori statici approssimano la terminazione rilevando ovvi loop infiniti. La decidibilità completa è impossibile; ogni strumento è necessariamente incompleto (Teorema di Rice).</p>
