<p><strong>Integer Linear Programming Infeasibility</strong> chiede: dato un sistema Ax ≤ b con vincoli interi, dimostrare che non esiste soluzione intera.</p><p>La versione di fattibilità di ILP è NP-completa; l'infeasibility è co-NP-completa. Il lemma di Farkas fornisce certificati di infeasibility polinomiali per il rilassamento reale, ma l'infeasibility intera può richiedere tagli di Chvátal-Gomory di dimensione esponenziale.</p>
