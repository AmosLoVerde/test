<p><strong>Esempio: sfida RSA-2048</strong></p><p>RSA-2048: un numero di 617 cifre n = p·q con p,q primi (~300 cifre ciascuno). Il certificato NP: p (fattore ≤ n^½). Verifica: dividere n per p in O((log n)²). Il certificato co-NP: la coppia (p,q) con testimoni di primalità per ciascuno.</p><p>Miglior algoritmo classico noto: GNFS, complessità euristica L[1/3, 1.923] ≈ exp(1.923·(log n)^{1/3}·(log log n)^{2/3}). Per RSA-2048: ≈ 2^112 operazioni — attualmente non fattibile.</p>
