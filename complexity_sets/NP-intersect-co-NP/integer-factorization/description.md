<p>La <strong>Fattorizzazione Intera</strong> chiede: dati n e k, n ha un fattore d con 1 < d ≤ k? Questa è in NP (certificato: il fattore d, verificabile per divisione) e in co-NP (certificato: la fattorizzazione prima completa, verificabile per moltiplicazione e test di primalità).</p><p>La fattorizzazione è la base della crittografia RSA. Nessun algoritmo classico polinomiale è noto; l'algoritmo quantistico di Shor risolve il problema in O((log n)³).</p>
