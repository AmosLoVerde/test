<p><strong>Esempio: fattorizzazione di Fermat per numeri vicini a un quadrato</strong></p><p>Se n = p·q con p,q vicini in valore, allora n ≈ ((p+q)/2)² - ((p-q)/2)². Si scrive n = a² - b² = (a-b)(a+b).</p><p>Algoritmo: provare a = ⌈√n⌉, ⌈√n⌉+1, ... finché a²-n è un quadrato perfetto b². Allora p=a-b, q=a+b. Tempo: O((p-q)/2 · polylog) — veloce quando p≈q. La sicurezza RSA richiede quindi che p e q siano scelti ben distanti.</p>
