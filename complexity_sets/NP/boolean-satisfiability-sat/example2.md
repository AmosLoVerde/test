<p><strong>Esempio: Circuit Satisfiability</strong></p><p>Dato un circuito booleano C con n input e un output, esiste un'assegnazione degli input che produce output 1?</p><p>Circuit-SAT è NP-completo (prova di Cook-Levin). Il certificato è il vettore di input. Verifica: valutare il circuito porta a porta in ordine topologico, O(|C|). È la base del Bounded Model Checking (BMC) nella verifica formale.</p>
