<p><strong>Esempio: Knight's Tour sulla scacchiera</strong></p><p>Il tour del cavallo chiede se un cavallo può visitare ogni casella di una scacchiera n×n esattamente una volta. Le caselle sono vertici, le mosse valide sono archi → trovare un cammino hamiltoniano.</p><p>Per n=5 esiste un tour chiuso. L'euristica di Warnsdorff trova soluzioni efficientemente in pratica, ma il problema decisionale generale rimane NP-completo per configurazioni arbitrarie.</p>
